# RepoLens

RepoLens is an AI-powered codebase intelligence platform that converts GitHub repositories into structured visual understanding using graph intelligence and LLM reasoning.

## Problem
Developers and students spend hours understanding new codebases.

## Solution
RepoLens automatically analyzes repositories and generates:
- Architecture diagrams
- Dependency graphs
- AI-generated explanations
- Learning paths for new developers

## Tech Stack
- AWS Bedrock
- Amazon Neptune
- AWS Lambda
- Amazon S3
- Python
